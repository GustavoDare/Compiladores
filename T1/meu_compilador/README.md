## Descrição do Projeto
Este projeto implementa o primeiro trabalho (T1) da disciplina Construção de Compiladores. Consiste na criação de um analisador léxico para a Linguagem Algorítmica (LA) utilizando ANTLR4 e Java.

## Pré-requisitos
Para garantir a correta compilação e execução, você precisará dos seguintes programas instalados:
* **Java JDK (versão 11 ou superior):** Necessário para compilar e executar a aplicação Java e o ANTLR4.
* **Apache Maven (versão 3.6+):** Utilizado para gerenciar dependências, executar a geração da gramática léxica e realizar o build do projeto (`.jar`).

## Passos para Compilação
Abra o terminal/linha de comando na raiz do projeto (onde o arquivo `pom.xml` está localizado) e execute o comando:
